# KoPapirGyikSpock

Egy egyszerű játék, az ismert kő papír olló kicsit fejlettebb változata.

Előkövetelmény:

        maven 3.x 
        jdk 1.8

Futtatás:

A projekt könyvtárból a következő parancsokkal lehet futtatni:

        Fordítás:
        
        >mvn package
        
        Futtatás:
        
        >java .jar target/KoPapirGyikSpock-1.0-jar-with-dependencies.jar
